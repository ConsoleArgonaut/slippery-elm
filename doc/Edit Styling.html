How to edit the style of a slippery elm page

Preconditions

The following conditions are needed to style a slippery elm page:
  • Slippery elm is installed
  • Access to the slippery elm files is given

    • Edit style

    To edit the page style follow the following steps:
  • Open the folder "Styling"
  • In this folder are all styling files which can be edited freely, like the base html and css
  • IMPORTANT: When making references to css and other files add "Styling/" before the reference

    • Placeholders

    When editing the index.html the following placeholders can be used:
  • [elm_PageContent] - The content of the page
  • [elm_PageNav] - The navigation menu of the page
  • [elm_Login_Link] - The link to login, or logout
  • [elm_Login_Text] - The text of the login link "Anmelden" or "Abmelden"
