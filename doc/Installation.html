Installation Manual

Preconditions

The following conditions are needed to install slippery elm:
  • Slippery elm installation files were downloaded
  • A Maria DB or Postgresql database is available
  • PHP 7 can be hosted

    • Installation

    To install slippery elm follow the follwing steps:
  • Unzip installation files to web folder
  • Set Read and Write permissions on web folder
  • Create empty database and database login for website
  • Colation of Database is utf8_general_ci
  • Set database connection string in config.php
  • Open website and create admin login
