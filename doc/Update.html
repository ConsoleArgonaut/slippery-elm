Update Manual

Preconditions

The following conditions are needed to install slippery elm:
  • Slippery elm update files were downloaded
  • Slippery elm already got installed

    • Update

    To update slippery elm follow the follwing steps:
  • Unzip update files to web folder
  • Set Read and Write permissions on web folder
  • Open the website and the update automatically gets executed in the background
